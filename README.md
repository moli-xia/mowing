# 绝地收割 (Final Mowing)

一款基于 WebGL 的僵尸围城射击游戏，玩家需要在僵尸潮中保卫自己的阵地。

## 游戏特性

- 🎮 **第一人称射击**：流畅的 WASD 移动 + 鼠标瞄准射击
- 🧟 **僵尸围城**：多波次僵尸进攻，难度递增
- 🌵 **沙漠战场**：精致的 3D 沙漠场景，包含多种障碍物
- 🎵 **音效系统**：背景音乐 + 射击/击杀音效
- ⚡ **性能优化**：针对多平台浏览器优化的渲染性能

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 瞄准 |
| 左键 | 射击 |
| R | 换弹 |
| ESC | 暂停 |

## 技术栈

- **Three.js** - 3D 渲染引擎
- **Vite** - 构建工具
- **Web Audio API** - 音频系统

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到服务器

1. 构建生产版本：`npm run build`
2. 将 `dist` 目录部署到 Web 服务器

## 系统要求

- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 支持 WebGL 2.0
- 键盘 + 鼠标
